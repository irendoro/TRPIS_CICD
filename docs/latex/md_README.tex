\hypertarget{md_README_autotoc_md1}{}\doxysection{Описание}\label{md_README_autotoc_md1}
Этот проект содержит несколько функций для математических вычислений, а также тестов для проверки их корректности. Включает три функции\+:


\begin{DoxyEnumerate}
\item {\ttfamily factorial(n)} — вычисляет факториал числа.
\item {\ttfamily add\+\_\+numbers(a, b)} — возвращает сумму двух чисел.
\item {\ttfamily is\+\_\+prime(n)} — проверяет, является ли число простым.
\end{DoxyEnumerate}

Кроме того, в проекте есть набор тестов с использованием библиотеки {\ttfamily unittest} для проверки корректности работы этих функций.\hypertarget{md_README_autotoc_md2}{}\doxysection{Функции}\label{md_README_autotoc_md2}
